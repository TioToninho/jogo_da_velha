# Jogo da Velha 

Jogo da velha usando Python 3. 

O jogo funciona dando os seguintes comando: 

| ce     | cm     | cd     |
| ------ | ------ | ------ |
| **me** | **mm** | **md** |
| **be** | **bm** | **bd** |

Só executar no terminal e se divirta !!!
